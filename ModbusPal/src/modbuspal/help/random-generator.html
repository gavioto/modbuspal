9.2 Random generator The Random generator creates random values in the interval defined by the specified minimum and maximum values. The minimum and maximum values are either absolute or relative. If relative, the actual minimum value is obtained by summing the current automation’s value (at the moment when this generator starts) and the value specified in the “Minimum value” input box. If relative, the actual maximum value is obtained by summing the current automation’s value (at the moment when this generator starts) and the value specified in the “Maximum value” input box.
