RXTX project

ModbusPal relies on the RXTX project in order to provide support for serial ports.

The RXTX communication API is made of a Java library and a native OS library. It offers the same interface as the Java Communications API. But, instead of Java Communications, all platforms are supported, primarily Windows and Linux.

If you don't have RXTX communication API installed, ModbusPal will only offer TCP/IP support.

Download and installation instructions: http://www.rxtx.org
