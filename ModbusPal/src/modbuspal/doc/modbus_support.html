Modbus support

Implementation

The follwing lists details what are the Modbus implementations supported by ModbusPal.

Encoding:

  • RTU
  • ASCII

Transport:

  • TCP/IP
  • Serial (requires RXTX Communication API)

Variants:

  • Modbus
  • J-Bus

Functions

The following list indicates what are the function codes supported by ModbusPal

Slaves (incoming requests):

  • 0x03: Read holding registers
  • 0x10: Write multiple registers

Master (outgoing requests):

  • None
